# 📊 Social Media Sentiment Analysis

## 📌 Overview
This project uses Machine Learning and NLP to analyze text and classify it into:
- Positive
- Negative
- Neutral

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- NLP (Text Cleaning)
- Plotly

## 🚀 Features
- Text preprocessing
- Sentiment prediction
- Machine learning model
- Visualization (pie chart)

## 📊 Output
The model predicts sentiment from input text.
